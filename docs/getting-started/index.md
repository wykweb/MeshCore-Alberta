# Get Started with Alberta MeshCore

This guide will help you join the MeshCore network in Calgary and Alberta.

## What you need

You will generally need:

1. A supported LoRa device
2. MeshCore companion firmware
3. A phone or computer for configuration
4. Confirmed regional radio settings
5. A suitable antenna
6. An understanding of companion and repeater roles

## Recommended process

1. Learn how MeshCore works
2. Choose a supported device
3. Flash companion firmware
4. Configure the device
5. Confirm the Alberta regional settings
6. Test your node
7. Join the local community
8. Consider contributing an observer or repeater

!!! warning "Do not guess radio settings"

    Use only settings published on the Alberta configuration page or provided
    by a confirmed regional coordinator.
