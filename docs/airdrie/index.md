# Airdrie MeshCore Network

## Overview

The Airdrie MeshCore network is part of the wider Alberta MeshCore community and operates within the Calgary regional zone.

Airdrie nodes use the **YYC** regional identifier so they can participate in the same regional infrastructure and community network as Calgary-area MeshCore users.

The local network can include:

* Companion nodes
* Portable MeshCore devices
* Home-based nodes
* Community repeaters
* MQTT observers
* Packet-analysis infrastructure

The goal is to improve MeshCore coverage throughout Airdrie while maintaining compatibility with the wider Calgary and Alberta network.

## Regional Zone

| Setting                  | Value                        |
| ------------------------ | ---------------------------- |
| Community                | Airdrie MeshCore             |
| Province                 | Alberta                      |
| Regional IATA identifier | **YYC**                      |
| Wider regional network   | Calgary and surrounding area |

Airdrie uses **YYC** because it is part of the Calgary regional operating area.

## Getting Connected

To join the Airdrie MeshCore network:

1. Use a supported LoRa device.
2. Install the appropriate MeshCore firmware.
3. Configure the Alberta radio settings.
4. Use **YYC** as the regional IATA identifier where required.
5. Test your node with nearby companions or repeaters.
6. Consider contributing a repeater or observer to improve local coverage.
