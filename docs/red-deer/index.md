# Red Deer MeshCore Network

## Red Deer MeshCore

Red Deer MeshCore supports the growth of community-operated LoRa mesh communication across Red Deer and Central Alberta.

The local network is part of the broader Alberta and Canadian MeshCore ecosystem and may include companion nodes, repeaters, observers, mobile mapping nodes, and other community infrastructure.

## Regional Information

## Getting Connected

To join the Red Deer MeshCore network:

1. Choose a supported LoRa device.
2. Install the correct MeshCore firmware.
3. Apply the Alberta radio settings.
4. Use **YQF** as the regional identifier where required.
5. Test your node with nearby companions or repeaters.
6. Review the repeater ID usage page before deploying a repeater.
7. Join the local Facebook group to connect with other users.

## Network Goals

The Red Deer MeshCore community can help support:

* Expanded coverage throughout Red Deer
* Better connectivity across Central Alberta
* Responsible repeater deployment
* Coverage mapping and testing
* Community participation
* Integration with the wider Alberta MeshCore network
