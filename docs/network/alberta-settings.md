# Alberta MeshCore Settings

This page contains verified settings used by participating Alberta MeshCore
communities.

!!! danger "Radio settings are still being verified"

    Do not copy radio parameters from another region or an unverified
    screenshot. Confirmed Alberta and Calgary settings will be published here
    following regional review.

## Confirmed general information

| Setting | Value |
|---|---|
| Country | Canada |
| Province | Alberta |
| Frequency band | 915 MHz regional hardware |
| Initial focus | Calgary and surrounding area |
| Calgary regional identifier | YYC |
| Network type | Community-operated MeshCore network |

## Settings requiring confirmation

The following will be published only after confirmation:

- Precise frequency
- Bandwidth
- Spreading factor
- Coding rate
- Transmit power guidance
- Shared channel names
- Public channel keys
- Repeater naming convention
- Observer topic structure
- MQTT broker information

## Verification policy

Each published setting should include:

- the setting value;
- the geographic area where it applies;
- who confirmed it;
- the date it was reviewed; and
- whether it is official, recommended or experimental.
