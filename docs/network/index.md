# Alberta MeshCore Network

Alberta MeshCore supports the development of community-operated LoRa mesh
networks throughout Alberta.

## Initial regional focus

The initial documented region is Calgary and surrounding communities,
including:

- Calgary
- Airdrie
- Cochrane
- Chestermere
- Okotoks
- High River

Additional community pages will be added when reliable local information is
available.
