# Alberta MeshCore Monitoring Tools

The Alberta MeshCore community has access to a growing collection of monitoring, mapping, packet-analysis, and visualization tools.

These services help users monitor network activity, analyze packet traffic, discover coverage, and view live MeshCore infrastructure across Alberta and Canada.

---

## Live Monitoring

### YYC MeshCore Beacon

Monitor the Calgary and YYC MeshCore network using the Beacon dashboard.

[Open YYC MeshCore Beacon](https://beacon.canadaverse.org/?tab=Home&iata=YYC){ .md-button .md-button--primary }

---

### YYC MeshCore Beacon LIVE

View live packet activity and network updates for the YYC MeshCore network.

[Open Beacon LIVE](https://beacon.canadaverse.org/?tab=Live&iata=YYC&csf=name){ .md-button .md-button--primary }

---

### Carto-Live Canadaverse

Interactive maps and operational views for the Canadaverse MeshCore infrastructure.

[Open Carto-Live](https://carto.canadaverse.org/){ .md-button .md-button--primary }

---

### Beacon MeshCore Calgary

View the Calgary MeshCore Beacon map.

[Open Beacon MeshCore Calgary](https://dev.meshcore.ca/?iata=YYC&tab=Map){ .md-button .md-button--primary }

## Observer and Packet Monitoring

### CoreScope MeshCore Canada Observers

View active MeshCore observers across Canada.

[Open CoreScope Observers](https://live.meshcore.ca/#/observers){ .md-button .md-button--primary }

---

### CoreScope MeshCore Canada LIVE

View active MeshCore activity across Canada from Coast to Coast via live.meshcore.ca.

[Open CoreScope MeshCore Canada LIVE](https://live.meshcore.ca/#/live){ .md-button .md-button--primary }

---

### CoreScope MeshCore YYC

View active MeshCore LIVE activity in Calgary & area.

[Open CoreScope YYC](https://corescopeyyc.meshmonitoring.com/#/live){ .md-button .md-button--primary }

---

### CoreComms.net Map and Packets

View MeshCore maps and packet activity.

[Open CoreComms.net](https://corecomms.net/){ .md-button .md-button--primary }

---

### MeshRank.net YYC Channel

Browse messages published on the YYC MeshCore channel.

[Open MeshRank YYC](https://www.meshrank.net/messages?channel=yyc){ .md-button .md-button--primary }

### WAEV.app

View the live MeshCore map centered on the Calgary and surrounding region.

### MeshCore MeshMapper
