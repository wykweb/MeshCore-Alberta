# Calgary MeshCore

Calgary is the initial launch region for Alberta MeshCore.

This section documents the local companion network, repeaters, observer
stations, MQTT infrastructure, dashboards and community projects serving
Calgary and surrounding areas.

## Calgary regional identifier

The identifier used for Calgary documentation and monitoring is: YYC
