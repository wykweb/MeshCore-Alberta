# Lethbridge MeshCore Network

## YQLMesh

**Connecting Lethbridge, One Node at a Time**

YQLMesh is the MeshCore community serving Lethbridge and the surrounding Southern Alberta region.

The Lethbridge MeshCore network is part of the wider Alberta and Canadian MeshCore community. It helps local users communicate over a long-range, low-power LoRa mesh network without depending entirely on cellular or internet infrastructure.

The local network may include:

* Companion nodes
* Portable MeshCore devices
* Home-based nodes
* Community repeaters
* MQTT observers
* Packet-analysis infrastructure
* Mobile coverage-mapping nodes

## Getting Connected

To join the Lethbridge MeshCore network:

1. Choose a supported LoRa device.
2. Install the appropriate MeshCore firmware.
3. Configure the Alberta radio parameters.
4. Use **YQL** as the regional IATA identifier where required.
5. Test your device with nearby companions or repeaters.
6. Visit the YQLMesh website for local community information.
7. Consider contributing a repeater, observer, or mobile coverage-mapping node.

## Network Goals

The goals of YQLMesh include:

* Expanding MeshCore coverage throughout Lethbridge
* Connecting users across Southern Alberta
* Supporting reliable community repeaters
* Mapping local coverage
* Encouraging community participation
* Connecting Lethbridge with Alberta and Canadian MeshCore infrastructure
